Top Research Insights
Membership UX
01
Upgrade friction is biggest drop-off
7/10 users couldn't identify the value proposition before hitting the paywall. CTA copy and benefit hierarchy need rework.
02
Podcast progress not persistent
Users expect playback position to resume across devices. 6/8 testers named this as a top frustration.
03
Admin search is broken for non-German names
Internationalized name search fails silently. Admins resort to scrolling manually through 2,000+ members.
04
Mobile member profile is unused
Only 12% of mobile users visit their profile. Key features buried 3 levels deep. Needs nav restructure.
