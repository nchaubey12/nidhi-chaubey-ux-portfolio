# UX Research Dashboard — Nidhi Chaubey

A dark-theme, SaaS-style UX/UI research dashboard designed for a 
language learning platform (Learn German / Learn Languages).

Built as a portfolio piece showcasing UX research workflows, sprint 
planning, user testing metrics, podcast player UI, and role-fit 
assessment — all in a single interactive interface.

## What's inside
- Sprint kanban board with real research tasks
- Podcast player feature prototype (UX focus)
- User testing & insights metrics
- Role fit skill assessment
- Research activity heatmap
- Admin interface & membership platform UX flows

## Built with
- Vanilla HTML / CSS / JavaScript
- Google Fonts (Syne + Instrument Sans)
- No frameworks · No dependencies
